# star-generator
A simple ES6 lib that draws stars in your project.

## Getting Started
Install lib in your project with npm
```
npm install star-generator
```
---
Import to use:
```js
import starGenerator from '~/star-generator';

starGenerator(screen.width, screen.height, 1);
```
